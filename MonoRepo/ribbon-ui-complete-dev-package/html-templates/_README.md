# HTML Templates Directory

This folder contains **28 ready-to-reference HTML templates** showing:
- Semantic HTML structure
- CSS class naming conventions
- CSS variable usage
- Responsive design patterns
- Accessibility attributes

## Quick Navigation

- **css-variables.css** - All CSS variables (colors, spacing, typography)
- **6 Major Components** - RibbonWindow, Ribbon, Toolbar, Sidebar, ButtonBar
- **21 Controls** - Button, Toggle, Input, Checkbox, Radio, etc.
- **INDEX.md** - Complete list of all templates

## How to Use

1. Find the template for the component you're implementing
2. Review the HTML structure and class names
3. Check the CSS variables used
4. Use this as reference when building your React component
5. Maintain the same semantic HTML structure and CSS naming

## Examples

- Building a Button? → Review `controls/button.html`
- Building RibbonWindow? → Review `ribbon-window.html`
- Need all CSS variables? → Review `css-variables.css`

All templates use the `.ribbon-{component}--{modifier}` naming convention.
