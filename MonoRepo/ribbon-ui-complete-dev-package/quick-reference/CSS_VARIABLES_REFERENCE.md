# CSS Variables Quick Reference

---

## Colors

```css
--ribbon-primary: #4472C4           /* Primary blue */
--ribbon-primary-hover: #3860A0
--ribbon-primary-active: #2E4C7E
--ribbon-primary-disabled: #B4C7E7

--ribbon-surface: #FFFFFF           /* Light background */
--ribbon-surface-variant: #F5F5F5
--ribbon-surface-hover: #EEEEEE
--ribbon-surface-active: #E0E0E0
--ribbon-surface-disabled: #F9F9F9

--ribbon-text: #262626              /* Text colors */
--ribbon-text-secondary: #595959
--ribbon-text-muted: #8A8A8A
--ribbon-text-disabled: #BFBFBF
--ribbon-text-inverse: #FFFFFF

--ribbon-success: #70AD47
--ribbon-warning: #FFC000
--ribbon-danger: #C5504A
--ribbon-info: #0563C1

--ribbon-border: #D0CECE
```

## Theme Overrides

### Dark Theme `[data-theme="dark"]`
```css
--ribbon-primary: #5B9BD5
--ribbon-surface: #2D2D2D
--ribbon-text: #FFFFFF
--ribbon-border: #464646
```

### Accessible `[data-theme="accessible"]`
```css
--ribbon-primary: #0052CC
--ribbon-border-width: 2px
--ribbon-font-size: 14px
```
