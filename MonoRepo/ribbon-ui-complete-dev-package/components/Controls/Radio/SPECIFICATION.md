# Radio Button Control Specification

**Control Name**: Radio Button  
**Category**: Selection (4)  
**Status**: ✅ Fully Specified

## Overview
Radio Button allows selection of one option from a mutually exclusive group.

## Props
- `id`: string
- `name`: string
- `label`: string
- `value`: string
- `checked`: boolean
- `onChange`: (value: string) => void
- `disabled`: boolean
